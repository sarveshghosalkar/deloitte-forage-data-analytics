# Deloitte-forage-data-analytics
Data analytics project based on Deloitte Forage simulation, analysing factory machine failures and employee equality using Excel and Tableau

Deloitte Forage Data Analytics Project

🔍 Project Overview

This project is based on the Deloitte Data Analytics Virtual Experience Program (Forage). It focuses on analyzing operational and HR data to generate actionable business insights for a manufacturing client, Daikibo.

🧩 Business Problem

The client collected large-scale machine telemetry data and employee compensation data to answer two key business questions:

🏭 Task 1: Machine Failure Analysis
Analyzed one month of telemetry data (May 2021) across 4 global factories:
Tokyo (Meiyo)
Osaka (Seiko)
Berlin
Shenzhen
Each factory operated 9 machine types sending data every 10 minutes
Objective:
Identify which factory experienced the highest number of machine breakdowns
Determine which machine types failed most frequently in that location

⚖️ Task 2: Equality Analysis
Worked with employee compensation data
Created a classification system based on Equality Score:
Fair (−10 to +10)
Unfair (−20 to −10 and 10 to 20)
Highly Discriminative (< −20 or > 20)
Implemented classification logic in Excel to highlight potential inequality across factories and job roles

🛠 Tools & Technologies
Microsoft Excel (data cleaning & classification)
Tableau (data visualization & dashboarding)
JSON (telemetry dataset)

📈 Key Outcomes
Identified high-risk factories with frequent machine breakdowns
Highlighted critical machine types requiring maintenance focus
Detected inequality patterns in employee compensation
Built a structured and visual approach to support decision-making

📊 Dashboard

The Tableau dashboard provides a visual summary of:

Machine breakdown distribution by factory
Machine performance trends
Equality score distribution across roles and locations

💡 Key Learnings
Handling real-world structured and semi-structured data
Transforming raw data into meaningful insights
Building dashboards for business decision-making
Applying analytical thinking to operational and HR problems

This project demonstrates practical data analytics skills aligned with real-world consulting scenarios.
